FrameFlow Design System

Dark-first B2B marketplace UI — for motion-graphics templates. Envato-level density and trustworthiness.

01 — Foundation

Colors

A deep, calm background palette keeps attention on the content. Accent colors mark only status and action — never decorative.

Surfaces
Accent & status
UX: Violet = brand/contributor action. Green = approval/success. Yellow = pending. Orange = soft reject. Red = hard reject/block. This mapping is the same across both dashboards.
