FAZA 5 · WEB APP REDESIGN · MOCKUP FLOW BOARD
FrameFlow — logged-in web app, two directions

Static design board for the getframeflow.app logged-in experience — Home, Stock Catalog, AI Toolkit and Account/Billing. Two genuinely different directions built on the same FrameFlow tokens and the real feature set (plans Free $0 · Pro $19 · Studio $59; credits ✦5 image / ✦40 video / ✦8 voice / ✦3 SFX; filters App·Category·Plan·Orientation·Quality). Nothing here touches the live platform.

Variant A — Editorial · media-forward Variant B — Workspace · utility-forward tokens: page #06080B · lime #C2F04A fonts: Hanken Grotesk + IBM Plex Mono icons: inline SVG only
VARIANT A
Editorial · media-forward

Artlist-like storefront feel. Horizontal top navigation, big greeting hero, immersive gradient cards, generous whitespace. The template detail is a dedicated page (not a modal), and the AI composer is a floating dock over a canvas — content is the hero, chrome stays out of the way.
