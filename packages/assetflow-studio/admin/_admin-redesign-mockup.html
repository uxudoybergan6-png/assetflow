YAGONA NAV — 4 GURUH
Bitta admin sidebar, 4 mantiqiy guruh: Ish maydoni (moderatsiya oqimi) · Boshqaruv (odamlar, tarif, xabar) · Biznes (moliya, narx, payout — YANGI) · Tizim (sozlama, log, audit). Ikkita parallel nav YO'Q.
SOXTA TUGMA YO'Q
Joriy admin'dagi o'lik/dekorativ tugmalar OLIB TASHLANDI. Har boshqaruv REAL API'ga ulanadi (approve/reject/block/plan/kredit/limit/broadcast/reindex) yoki "keyingi versiya" deb ochiq belgilanadi. Halol bo'sh holatlar (navbat bo'sh, obunachi yo'q).
YANGI: BIZNES MARKAZ

"RASVO" LAYOUT TUZATILDI
Joriy admin'ning tarqoq/zich bo'lmagan tuzilishi qayta ishlangan: barqaror jadval tili, ustun tekislash (tabular-nums), stat-karta gridi, ikki-panelli moderatsiya/xabar, izchil badge ranglari. Ranglar faqat token — hardcode yo'q.
