# Design reference — Claude Design eksportlari

Bu papka — `claude.ai/design`'dan eksport qilingan dizayn prototiplari (vizual referens).
Bular **manba kod emas**, balki Claude Code plaginga ko'chirishda qaraydigan namuna.

## Fayllar
- `AssetFlow-Artlist.html` — Artlist uslubidagi flat (near-black) variant.
- `AssetFlow-LiquidGlass.html` — Apple Liquid Glass varianti (shaffof shisha).

## Foydalanish
1. Claude Design → Export → Standalone HTML → Download.
2. Yuklab olingan faylni shu papkaga ko'chiring (yuqoridagi nom bilan).
3. Claude Code'ga: "docs/design-reference/<fayl> ni o'qi — vizual referens.
   AssetFlow_Plugin.html (vanilla HTML + tokens.css) ga moslab qayta qur."

> Eslatma: bu fayllar Tailwind/React bo'lishi mumkin — to'g'ridan plaginga ko'chirilmaydi,
> faqat ko'rinish namunasi sifatida ishlatiladi.
